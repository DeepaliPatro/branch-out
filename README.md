# Branch Out

This project is a fullstack e-commerce application for a garden center.

# Tech Stack

## Server Side

- MongoDB Atlas and Mongoose
- NodeJS
- Express
- bcrypt js
- jsonwebtoken
- Typescript

## Client Side

- Angular
- Typescript
- SCSS
- Angular Material
- Ngx Bootstrap
- Angular FontAwsome

# Features

- Search by name
- Find by Tag
- Click on a card to see all details
